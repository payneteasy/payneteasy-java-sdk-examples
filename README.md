# Payneteasy Java SDK Examples

A collection of ready-to-run Java examples demonstrating how to integrate with the [Payneteasy Payment Gateway](https://doc.payneteasy.eu/) using the Payneteasy Java SDK.

## Examples

| Example                                 | Description                                                                                       |
|-----------------------------------------|---------------------------------------------------------------------------------------------------|
| [sale-form-example](sale-form-example/) | Initiates a Sale Form payment and returns a redirect URL for the customer to complete the payment |

## Prerequisites

- Java 21+
- Maven 3.8+
